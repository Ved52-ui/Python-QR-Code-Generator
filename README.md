## 🎯 What This App Does

This application allows you to create **high-quality UPI QR Codes** with your:
- ✅ Payee Name (Shop / Business Name)
- ✅ UPI ID (VPA)
- ✅ Fixed amount (₹)
- ✅ Payment Remark (Optional)
- ✅ Transaction Reference (Optional)

Once generated, you can:
- 💾 Save QR Image (PNG - Print Ready)
- 📋 Copy UPI Payment Link
- 🌐 Test the link in your UPI App
- 🔄 Switch GUI Themes (Light / Dark / System)

## 🌟 Features (Detailed)

- Sleek, modern UI using **CustomTkinter**
- Works **offline** (no internet needed)
- Generates **high-error-correction QR Codes**
- Automatic **caption included** for printing:
  - Shop Name
  - Amount
  - Remark (if added)
- **Multiple Themes**:
  - Light
  - Dark
  - Auto (System Mode)
- **Change Accent Colors**:
  - Blue
  - Green
  - Dark Blue
